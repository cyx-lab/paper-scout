### Problem

- **pdf_url:** https://arxiv.org/pdf/2604.26610v1
- **summarized_at:** `2026-04-30T09:58:51`

### Related topics

quasicrystal:aperiodic

### One-sentence summary

This paper introduces the momentum space linking number (MoLiN) to characterize and classify the topology of infinitely extending vector wave fields that lack a global invariant in position space.

### Problem

Traditional position-space topological invariants (like Skyrmion numbers) require a uniform background or periodicity, making them inapplicable to infinitely extending, aperiodic vector wave fields where global classification fails.

### Approach

The authors map wave fields to momentum space via Fourier transform, define the MoLiN to describe the winding of phasors around the momentum ring (representing a Berry phase), and validate its robustness using surface plasmon and water wave experiments.

### Main takeaway

The MoLiN is a global topological invariant that is highly robust against spatial deformations and provides a unified description of topological charges for periodic, quasicrystalline, and aperiodic vector wave fields.

### Why it matters

It establishes a universal topological classification framework for complex vector fields across optics, acoustics, and fluid dynamics, with potential applications in topologically protected data storage and transmission.

### Value assessment

- **Major advance**
- It addresses the long-standing lack of a global topological description for infinitely extending aperiodic fields and demonstrates interdisciplinary universality.

### One-sentence summary

The study reveals that magnetotunnel conductance at a Weyl semimetal–layered Chern insulator junction saturates to a universal value at high magnetic fields due to topological constraints.

### Problem

To explore how the topological mismatch between a gapless Weyl semimetal and a layered Chern insulator affects interface states and magnetotransport properties.

### Approach

Using a two-band lattice model, the authors analyze the interface spectrum via exact diagonalization and simulate magnetotunnel conductance under a perpendicular magnetic field using KWANT.

### Main takeaway

Magnetotunnel conductance increases linearly at low fields but saturates to a universal constant beyond a critical field; this value depends only on system size and Chern layers, independent of interface coupling or Fermi-arc geometry.

### Why it matters

It identifies a transport mechanism governed by topological charge pumping rather than magnetic breakdown, providing a universal experimental signature for identifying topological semimetal heterojunctions.
